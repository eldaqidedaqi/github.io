# Apuestas Digitales de la Liga Española de Futbol

## Descripción
Aplicación web interactiva que permite a los usuarios participar en una apuesta de fútbol de la Liga Española. Ofrece modos de apuesta virtual y real, con premios basados en aciertos, múltiples idiomas, visualización de patrocinadores, y gestión de temporadas y apuestas.

## Características
- Apuestas con 15 partidos oficiales.
- Selección de pronósticos (1, X, 2) por partido.
- Modos Virtual (free game test) y Real (pagos seguros).
- Traducción multilenguaje con menú para cambiar idioma.
- Reloj digital en cabecera.
- Visualización de caja con bote de premios.
- Panel de patrocinadores con logos cuidados.
- Gestión de próximas temporadas y bloqueo de apuestas tras inicio.

## Instalación
1. Clona o descarga este repositorio.
2. Abre el archivo `index.html` en cualquier navegador moderno.
3. No requiere servidor para modo virtual, para modo real se recomienda backend adicional.

## Uso
- Marca el resultado esperado para cada partido.
- Pulsa "Continuar" cuando estén marcados los 15 pronósticos.
- Elige modo Virtual o Real para jugar.
- En modo Real, sigue instrucciones para introducir correo y método de pago.
- Consulta el panel de resultados para ver premios simulados y estado de apuestas.

## Modos de Juego
- **Virtual:** Apuesta simulada con 10 Dh virtuales. Muestra supuestas ganancias según aciertos.
- **Real:** Apuesta con dinero real, con integración para Google Pay, PayPal o Visa (simulado). Se registra apuesta y se suma a la caja común.

## Premios
- Acertar 11 pronósticos: 10 Dh
- Acertar 12 pronósticos: 30 Dh
- Acertar 13 pronósticos: 150 Dh
- Acertar 14 pronósticos: entre 2,000 y 8,000 Dh
- Acertar 15 pronósticos: bote entre 100,000 y 500,000 Dh
